# Northstar Design System

## Direction

The selected concept is Profile Studio: a public profile/link hub builder as the primary workflow, supported by CRM signal, analytics, and AI follow-up suggestions.

## Tokens

- Background: true white `#ffffff`.
- Muted surface: `#f8fafc`.
- Border: `#e2e8f0`.
- Text: slate/graphite, led by `#0f172a`.
- Accent: cobalt blue `#2563eb`, with teal/emerald used sparingly for positive analytics.
- Radius: 6-12px, with 8px as the default control radius.
- Motion: 160-240ms ease, only for feedback, reveal, and state changes.

## Component Rules

- Cards frame standalone modules only. Lists stay as grouped surfaces with row separators.
- Icons use Lucide outline style at 16-20px with consistent optical weight.
- Buttons are icon-capable and keyboard-focus visible.
- Tables, rows, switches, segmented controls, and swatches share the same border and focus language.
- App chrome uses compact type; marketing sections use larger spacing without oversized decorative effects.
